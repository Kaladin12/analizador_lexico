# analizador_lexico
Gramatica LL(1) generada para el nalizador sintÃ¡ctico del proyecto final de Compiladores.
## Variable inicial
```
MAIN -> EXPR | ASIGN | ESTRUCT
```
## Expresiones
```
EXPR ::= VALOR NEW_OPER
EXPR ::= AGR_OP EXPR AGR_CP NEW_OPER
NEW_OPER::= ARI_MD OP_T UP_OPER
NEW_OPER::= ARI_PS EXPR
NEW_OPER::= OPLOG OP_T UP_OPER
NEW_OPER::= OPREL EXPR
NEW_OPER::= ''
UP_OPER ::= ARI_PS EXPR
UP_OPER ::= OPREL EXPR
UP_OPER ::= ''
OP_T ::= VALOR LOW_OPER
OP_T ::= AGR_OP EXPR AGR_CP LOW_OPER
LOW_OPER::= ARI_MD OP_T
LOW_OPER::= OPLOG OP_T
LOW_OPER::= '' 
```
## Estructuras
```
ESTRUCT ::= COND
ESTRUCT ::= LOOP
ESTRUCT ::= INTER
ESTRUCT ::= LLAMA_AUX
COND ::= COND_IF IF_CONTENT TEMP
TEMP ::= ''
TEMP ::= COND_ELSE CONTENT
IF_CONTENT ::= AGR_OP EXPR AGR_CP CONTENT
CONTENT ::= AGR_OB MAIN AGR_CB 
INTER ::= INTERRUPTOR AGR_OP EXPR AGR_CP AGR_OB CASE_ AGR_CB
CASE_ ::= CASO AGR_OP VALOR AGR_CP  AGR_OB MAIN ROMPER AGR_CB CASE_END
CASE_END ::= CASE_ 
CASE_END ::= DEFAULT 
DEFAULT ::= PREDETERMINADO AGR_OB MAIN ROMPER AGR_CB
LOOP ::= LOOP_FOR_ABS 
LOOP ::= LOOP_WHILE 
LOOP ::= LOOP_DO 
LOOP_FOR_ABS ::= CADA AGR_OP FORS
FORS ::= LOOP_PY ID
FORS ::= LOOP_FOR
LOOP_FOR ::= ASIGN EOL EXPR EOL EXPR AGR_CP  CONTENT
LOOP_WHILE ::= MNTRS  AGR_OP EXPR CONTENT EOL
LOOP_DO ::= HAZ_MNTRS CONTENT EXPR EOL
LOOP_PY ::= EN RANGO AGR_OP EXPR AGR_CP AGR_CP CONTENT
ASIGN ::= TIPO ID OP_ASIG EXPR EOL 
```
## Axiliares
```
LLAMA_AUX ::= SLEEP 
LLAMA_AUX ::= PRINT 
LLAMA_AUX ::= LEN
SLEEP ::= SLP AGR_OP VALOR AGR_CP
PRINT ::= IMPR AGR_OP EXPR AGR_CP
LEN ::= LONGITUD AGR_OP VALOR AGR_CP
```
